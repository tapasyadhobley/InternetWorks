IP Addresses, Packets, and Routing

Posted by Tapasya Dhobley on October 14, 2020

IP Addresses

IP Address stands for Internet Protocol address. This is a numerical label that is assigned to each device that connects to a network.

These addresses are used for identification and location addressing.

These work to establish a path to the host.

There are two different versions of the Internet Protocol today. One of them is ARPANET which is the older version which was first launched in 1983. The second version is the Internet Protocol version 4 which is newer.

Packets

A packet is a collection of data that computers use to communicate with each other on the network.

They allow computers to be able to send and recieve information quick and efficiently.

Routing

Routing is the process of moving a packet from the source to the destination. This is performed by the router and it enables messages to be passed from one device to another.
