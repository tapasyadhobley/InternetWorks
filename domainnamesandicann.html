Domain names and Icann

Posted by Tapasya Dhobley on October 14, 2020

Domain Names

A domain name is the part of the network that identifies its belonging to a particular domain.

They are used to identify one or more IP addresses. Every domain name has a suffix that indicates the top level domain that it belongs to.

For example :

  • .gov
  • .edu
  • .org
  • .com
  • .net

The internet is based on IP addresses. Therefore, every web server requires a Domain Name System to translate domain names to IP addresses.

ICANN

ICANN stands for the Internet Corporation for Assigned Names and Numbers. This is a nonprofit organization that maintains the procedures of databases. This group ensures that the network is stable and secure.

Because the internet grew rapidly and widely, the Department of Commerce established this organization to follow certain policies.

One of the policies that was created is the Uniform Domain-Name Dispute Resolution Policy. This policy was put in place to create a quick resolution for domain name conflicts. This is helpful because it avoids the court system and allows for domain name disputes within ICANN.
